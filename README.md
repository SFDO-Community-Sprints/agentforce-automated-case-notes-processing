# agentforce-automated-case-notes-processing
# Project "Truffle Hog"

## About this Project
The goal of this project is to create an Agent-driven augmentation of NonProfit Cloud Case Management which parses Interaction Summaries to generate theme "tags" and follow up tasks.

## Project Progress Summary

### Project Start - Agentforce Sprint in SFO - December 2025
The initial idea for this project emerged from the ideation exercise at the beginning of this Sprint.  The goal is to speed processing of large volumes of text information inputting during the course of providing services into two major actionable/usable results:
* Tags that surface general themes
* Tasks on the Program Enrollment for follow up actions
